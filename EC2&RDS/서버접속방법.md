2. RDS(MySQL) 접속 - CLI

EC2 서버가 우분투라서 우분투 명령어 써야해요.

> $ mysql -u admin -p --host ssadola.caaqegau8sq4.ap-northeast-2.rds.amazonaws.com

이 명령어를 치면 mysql DB에 로그인 시도
