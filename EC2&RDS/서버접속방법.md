```
RDS 엔드 포인트 ssadola.****.rds.amazonaws.com

13.124.83.**


db 관리자 계정 admin/****
```

1. EC2 접속

키 페어(aws_key.pem) 필요!!!

Linux / Mac OS
> $ ssh -i aws_key.pem ubuntu@13.124.83.**

2. RDS(MySQL) 접속 - CLI

EC2 서버가 우분투라서 우분투 명령어 써야해요.

> $ mysql -u admin -p --host ssadola.caaqegau8sq4.ap-northeast-2.rds.amazonaws.com

이 명령어를 치면 mysql DB에 로그인 시도
